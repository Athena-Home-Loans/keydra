# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| v0.10   | :white_check_mark: |
| <= v0.9 | :x:                |

## Reporting a Vulnerability

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, please report them via email as described in [https://www.athena.com.au/security.txt](https://www.athena.com.au/security.txt). 
If possible, encrypt your message with our PGP key.
